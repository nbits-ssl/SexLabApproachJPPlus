直したいバグ

仕様
・NPC⇔NPCの時の断る場合はMCM倍率は機能しない。構造上の問題なのであまり直す気ない。

・PCへのお誘いをキャンセルで終了させるともう一回聞かれることがある。
　まあ、無視してたらしつこく聞き直されたということで。
・PCと話しているNPCをNPC⇔NPCで横取りされることがある。
　=> イベントのシーン進行中は対策済み、普通の会話では相変わらず横取り可能性あり。
